## ポイント
- Vue.jsとFirebaseを使用した認証機能つきチャットアプリです。
- デザインまわりはVuetifyを使用して作成しました。
- サイドバーから参加済みのトークルームにアクセスできます。
- サイドバーのJoinボタンを押し、ルームIDとパスワードを入力することで他の人が作成したルームに参加できます。
- サイドバーのNewボタンを押し、ルーム名、ID、パスワードを設定することで、新しいルームを作成できます。
<br>
